## 🎓 Input Format (Both Versions)

### backlog.json
```json
[
  {
    "epic": "Authentication",
    "feature": "User Login",
    "tshirt_size": "M",
    "roles": ["Fullstack", "QA"]
  },
  {
    "epic": "Dashboard",
    "feature": "Analytics",
    "tshirt_size": "XL",
    "roles": ["Fullstack", "UX", "QA"]
  }
]
```

### config.json (optional)
```json
{
  "hoursPerDay": 6,
  "sprintLengthWeeks": 2,
  "unitTestingPercentage": 15,
  "bugFixingPercentage": 20,
  "documentationPercentage": 10,
  "contingencyPercentage": 15,
  "startDate": "2025-11-03"
}
```

### T-Shirt Sizes
- **XS:** 9 hours (trivial)
- **S:** 18 hours (simple)
- **M:** 36 hours (moderate)
- **L:** 72 hours (complex)
- **XL:** 108 hours (very complex)
- **XXL:** 144 hours (major)
- **XXXL:** 189 hours (epic)

### Roles
Fullstack, QA, DevOps, BA, SM, UX
