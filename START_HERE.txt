OPTION 1: PYTHON VERSION (SIMPLEST)
================================================================

Requirements: Python 3.7+ only

Quick Start (Windows):
  1. cd standalone-python
  2. Double-click: RUN_EXAMPLE.bat
     (or run: python estimator.py --example)
  3. Double-click: RUN_EXAMPLE.bat again
     (or run: python estimator.py)

Quick Start (Mac/Linux):
  1. cd standalone-python
  2. python3 estimator.py --example
  3. python3 estimator.py

Files Created:
  - estimation.json (results)
  
Export to CSV:
  python estimator.py --csv
  (creates: estimation.csv, gantt.csv)

Documentation:
  standalone-python/README.md

================================================================
OPTION 2: NODE.JS VERSION (FULL-FEATURED)
================================================================

Requirements: Node.js 18+ with npm

Quick Start (Windows):
  1. cd migration-package
  2. Double-click: SETUP.bat
  3. Done! It auto-generates examples and runs

Quick Start (Mac/Linux):
  1. cd migration-package
  2. chmod +x SETUP.sh
  3. ./SETUP.sh

Manual Setup:
  1. cd migration-package
  2. npm install
  3. npm run estimate -- --example
  4. npm run estimate

Export to CSV:
  npm run estimate -- --csv

View AI Prompts:
  npm run estimate -- --prompts

Documentation:
  migration-package/README.md
  migration-package/QUICKSTART.md

================================================================
MIGRATION TO ANOTHER COMPUTER
================================================================

Python Version:
  1. Copy entire "standalone-python" folder
  2. On new computer: python estimator.py --example
  3. Done!

Node.js Version:
  1. Copy entire "migration-package" folder
  2. On new computer: Run SETUP.bat (Windows) or SETUP.sh (Linux/Mac)
  3. Done!

Complete Migration Guide:
  MIGRATION_GUIDE.md

================================================================
INPUT FILE FORMAT (BOTH VERSIONS)
================================================================

Create backlog.json:
[
  {
    "epic": "Authentication",
    "feature": "User Login",
    "tshirt_size": "M",
    "roles": ["Fullstack", "QA"]
  }
]

T-Shirt Sizes:
  XS   = 9 hours   (trivial)
  S    = 18 hours  (simple)
  M    = 36 hours  (moderate)
  L    = 72 hours  (complex)
  XL   = 108 hours (very complex)
  XXL  = 144 hours (major)
  XXXL = 189 hours (epic)

Roles:
  Fullstack, QA, DevOps, BA, SM, UX

================================================================
OUTPUT
================================================================

Both versions create:
  - estimation.json (or .csv)
  - Console summary with costs, timeline, team
  - Gantt data (CSV format with --csv flag)

Console shows:
  - Total cost
  - Duration (weeks, sprints)
  - Timeline (start/end dates)
  - Role efforts (hours per role)
  - Team composition (headcount)
  - Assumptions

================================================================
CUSTOMIZATION
================================================================

Hourly Rates:
  Python:  Edit ROLE_RATES in estimator.py (line 31)
  Node.js: Edit src/config.ts (line 11)

T-Shirt Hours:
  Python:  Edit TSHIRT_HOURS in estimator.py (line 21)
  Node.js: Edit src/config.ts (line 4)

Multipliers (both):
  Edit config.json:
    - contingencyPercentage: 15
    - bugFixingPercentage: 20
    - unitTestingPercentage: 15
    - documentationPercentage: 10

================================================================
TROUBLESHOOTING
================================================================

Python: "python is not recognized"
  Solution: Install from https://www.python.org/
  Or try: python3 instead of python

Node.js: "npm is not recognized"
  Solution: Install from https://nodejs.org/

Estimates too high/low:
  Adjust multipliers in config.json
  Or change T-shirt hour mappings
